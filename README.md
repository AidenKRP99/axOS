# KRP OS
The new KRP Studios operating system
## Specs of KRP OS
Linux base: Ubuntu
Desktop enviroment: GNOME
Bootloader: GRUB
## System Requirements
64-bit processor
4GB of RAM
30GB of hard disk space
1024x768 or bigger resolution
USB port for installation
